## Project Overview
This project analyzes end-to-end recruitment data to identify key hiring metrics such as Time-to-Hire, Offer Conversion Rate, and Source Effectiveness.

## Dataset
- Candidate ID
- Job Role
- Application Date
- Interview Date
- Offer Date
- Status (Joined / Rejected / Interviewed)
- Skills
- Hiring Source

## Key Metrics
- Time to Hire
- Joining Ratio
- Source-wise Hiring Effectiveness
- Role-wise Hiring Trends

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- GitHub
